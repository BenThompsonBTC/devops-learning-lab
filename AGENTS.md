# DevOps Learning Lab - Codex Instructions

## Repo Purpose

This repo is Ben's DevOps learning trail. Prioritize hands-on labs, clear notes, and portfolio-quality examples.

Good additions include:

- GitHub Actions workflows
- Docker examples
- Linux command notes
- CI/CD experiments
- Azure CLI or cloud automation labs
- Monitoring/logging demos
- Terraform later, once basics are stronger

Every lab should include:

- Goal
- Commands used
- What happened
- What Ben learned
- How to verify it worked

Keep examples small, practical, and easy to revisit.
